## 🎯 시스템 개요

### 시스템명
**취약점 관리 시스템 (Vulnerability Management System)**

### 개발 목적
- **자동화된 CVE 취약점 정보 수집**: GitHub 및 공식 CVE API에서 최신 보안 취약점 자동 수집
- **AI 기반 분석**: Google Gemini Pro를 활용한 POC(공격 코드) 자동 분석
- **실시간 알림**: 카카오톡을 통한 신규 취약점 실시간 전송
- **통합 관리**: 웹 기반 대시보드로 취약점 정보 통합 관리

### 시스템 특징
1. **완전 자동화**: 사람 개입 없이 24시간 자동 수집 및 분석
2. **AI 분석**: 보안 전문가 수준의 자동 분석 (공격 단계, MITRE 매핑, Snort 룰 생성)
3. **실시간 알림**: 새로운 취약점 발견 즉시 보안팀에 알림
4. **외부 연동**: API 토큰 기반 외부 시스템 연동 지원
5. **웹 대시보드**: 직관적인 웹 인터페이스로 누구나 쉽게 사용

## 🏆 주요 성과

### 1. 자동화 수집 시스템
- ✅ **GitHub CVE 저장소**: Python 언어 기반 POC 코드 자동 수집
- ✅ **공식 CVE API**: 상세 취약점 정보 자동 수집
- ✅ **파일 다운로드**: ZIP 파일 자동 다운로드 및 분류 저장
- ✅ **번역 기능**: README 및 설명 자동 한국어 번역

**수집 데이터 예시**:
- CVE 코드, 제목, 작성자, GitHub 링크
- CVSS 점수, 위험도, 영향받는 제품
- 공격 벡터, CWE 분류, 해결 방안

### 2. AI 기반 자동 분석
- ✅ **POC 코드 분석**: Gemini Pro를 통한 공격 코드 심층 분석
- ✅ **공격 단계 분류**: 각 공격 단계별 상세 설명 자동 생성
- ✅ **MITRE ATT&CK 매핑**: 국제 표준 프레임워크 자동 매칭
- ✅ **Snort 룰 생성**: IDS/IPS 탐지 규칙 자동 생성
- ✅ **대응 방안 제시**: 실무 적용 가능한 보안 조치 자동 생성

**분석 결과 예시**:
- 취약점 요약 (200-400자)
- 공격 단계별 상세 분석 (5-10단계)
- 패킷 샘플 및 예상 응답
- MITRE Tactic & Technique
- Snort 탐지 룰

### 3. 웹 기반 관리 시스템
- ✅ **대시보드**: 실시간 통계 및 시각화
- ✅ **CVE 목록/상세**: 수집된 취약점 정보 조회
- ✅ **POC 상세**: AI 분석 결과 조회
- ✅ **커뮤니티**: 자유게시판, 실시간 채팅
- ✅ **관리자 기능**: 사용자 관리, 권한 부여, 로그 확인
- ✅ **DB 조회**: 분석가를 위한 직접 DB 쿼리 기능

### 4. 외부 시스템 연동
- ✅ **API 토큰 관리**: 외부 시스템 인증 토큰 발급/관리
- ✅ **데이터 Export API**: REST API로 통합 데이터 제공
- ✅ **Python 수집 모듈**: 외부 PC에서 데이터 자동 수집
- ✅ **카카오톡 연동**: Intent 제어 방식 메시지 전송 지원

## 🎨 주요 기능

### 1️⃣ 자동 수집 기능

#### GitHub POC 수집
- **대상**: GitHub에서 "CVE-{년도} language:python" 검색
- **수집 항목**:
  - CVE 코드, 제목, 작성자
  - GitHub 링크, README
  - 수집 시간
- **파일 저장**: `CVE/CVE-XXXX-XXXXX/1,2,3.../` 구조로 ZIP 저장
- **번역**: README를 한국어로 자동 번역
- **중복 방지**: 동일 링크는 최대 5개까지만 저장

#### CVE 상세 정보 수집
- **대상**: cve.circl.lu API
- **수집 항목**:
  - 상태, 예약/공개/업데이트 날짜
  - 영향받는 제품 및 버전
  - CVSS 점수, 벡터, 위험도
  - CWE 분류, 공격 유형
  - 설명 및 해결 방안 (번역)
- **저장**: DB 저장 + 각 ZIP 폴더 내 `CVE_Info.txt` 저장

### 2️⃣ AI 분석 기능

#### Gemini Pro 분석
- **모델**: gemini-2.5-pro
- **입력**: POC 코드 전체 파일 + CVE_Info.txt
- **분석 내용**:
  1. **취약점 요약**: 취약점 유형, 영향, 공격 메커니즘
  2. **공격 단계**: 각 단계별 상세 분석
     - 단계 번호, 단계명
     - 상세 설명 (목적, 기술적 행위, 예상 동작, 보안 영향)
     - 패킷 샘플 (실제 HTTP/네트워크 패킷)
     - 예상 응답 (성공 시 서버 응답)
     - MITRE Tactic & Technique
     - Snort 탐지 룰
  3. **대응 방안**: 즉시 대응, 패치 정보, 완화 방법, 장기 대책

#### 자동 실행
- **주기**: 10분마다 자동 실행
- **대상**: AI_chk = 'N'인 POC만 분석
- **완료 처리**: 분석 완료 후 AI_chk = 'Y'로 업데이트
- **에러 처리**: 
  - Rate Limit: 10분 대기 후 재시도
  - Quota 초과: 다음 날까지 대기

### 3️⃣ 웹 대시보드

#### 홈 (대시보드)
**전체 통계**:
- 총 CVE 수, 총 POC 수
- AI 분석 완료/대기 건수
- 최근 7일 수집 현황 (그래프)

**시각화 차트**:
- CVSS 위험도 분포 (원형 차트)
- 공격 단계별 분석 (막대 차트, TOP 5)
- CWE 유형 (막대 차트, TOP 5)
- 공격 유형별 분석 (막대 차트, TOP 5)
- 영향받는 제품 (막대 차트, TOP 5)

**클릭 가능**:
- 각 통계 클릭 시 해당 CVE 목록으로 이동
- "더보기" 버튼으로 전체 목록 확인

**최근 CVE**:
- 최신 수집된 CVE 20개 표시
- CVE 코드, 위험도, 수집 시간
- 클릭 시 상세 페이지 이동

#### CVE 정보
**목록 페이지**:
- 검색: CVE 코드 또는 제품명
- 필터: 위험도 (CRITICAL/HIGH/MEDIUM/LOW), 상태 (PUBLISHED/REJECTED)
- 페이지네이션: 10/25/50/100개씩 보기
- 정렬: 수집 시간 최신순

**상세 페이지**:
- CVE 기본 정보 (코드, 상태, 날짜)
- 취약점 설명 (한국어)
- CVSS 정보 (점수, 벡터, 복잡도)
- 영향받는 제품 및 버전
- CWE 분류, 공격 유형
- 해결 방안 (한국어)
- 관련 POC 목록 (최대 10개)

#### POC 상세
**POC 정보**:
- GitHub 저장소 정보
- 제목, 작성자, 링크
- README (원문 + 번역)
- 수집 일시

**AI 분석 결과** (AI_chk = 'Y'인 경우):
- 취약점 요약
- 공격 단계별 상세 분석
  - 각 단계 펼침/접기 가능
  - 패킷 샘플 (코드 블록)
  - 예상 응답 (코드 블록)
  - MITRE ATT&CK 정보 (클릭 시 팝업)
  - Snort 룰 (코드 블록)
- 대응 방안

**관련 POC**:
- 동일 CVE의 다른 POC 최대 10개

### 4️⃣ 커뮤니티 기능

#### 자유게시판
- 글 작성/수정/삭제
- 파일 첨부 (이미지, 문서, 압축 파일)
- 파일 보안 검증 (악성 파일 차단)
- XSS 방지 (DOMPurify)
- 검색 (제목, 내용, 작성자, 전체)
- 페이지네이션
- 권한:
  - 본인 글: 수정/삭제 가능
  - 관리자: 모든 글 수정/삭제 가능

#### 실시간 채팅
- Socket.IO 기반 실시간 통신
- 닉네임 표시
- 최근 3일 메시지 보관
- 접속자 수 표시

### 5️⃣ 사용자 관리

#### 회원가입
- 필수 정보: 이름, 전화번호, 이메일, 아이디, 비밀번호, 닉네임
- 이메일 인증: 6자리 코드 (3분 유효)
- 비밀번호 암호화 (bcrypt)

#### 로그인
- JWT 토큰 기반 인증
- 로그인 로그 기록 (`wets.txt`)

#### 비밀번호 재설정
- 이메일 인증 필수
- 3단계 프로세스:
  1. 본인 확인 (아이디 + 이메일)
  2. 인증 코드 입력 (6자리, 3분 유효)
  3. 새 비밀번호 설정

#### 내 프로필
- 개인 정보 조회
- 닉네임 변경
- 비밀번호 변경 (현재 비밀번호 확인 필수)

### 6️⃣ 관리자 기능

#### 사용자 관리
- 전체 사용자 목록
- 권한 변경: user(일반) / analyst(분석가) / admin(관리자)
- 닉네임 변경
- 비밀번호 초기화
- 계정 삭제 (본인 계정 제외)

#### DB 조회 (analyst, admin)
- 테이블 선택: Github_CVE_Info, CVE_Info, CVE_Packet_AI_Analysis
- 필드별 검색
- 읽기 전용 (보안)
- SQL Injection 방지 (파라미터화 쿼리)

#### 로그 확인 (admin)
- 시스템 로그 파일 조회
- 에러 추적
- 접속 기록

#### API 토큰 관리 (admin)
- 토큰 생성: 이름, 만료일, 권한 설정
- 토큰 목록: 상태, 마지막 사용 시간
- 활성/비활성 전환
- 토큰 삭제

### 7️⃣ 외부 연동 기능

#### API Export
- **엔드포인트**: `GET /api/export/cve`
- **인증**: API 토큰 (X-API-Token 헤더)
- **필터**:
  - `cve`: 특정 CVE 코드
  - `severity`: 위험도 (CRITICAL, HIGH, MEDIUM, LOW)
  - `from_date`: 수집 시작 날짜
  - `page`: 페이지 번호
  - `limit`: 페이지 크기
- **응답**: 통합 JSON 데이터
  - POC 정보 (GitHub)
  - CVE 정보 (상세)
  - AI 분석 (모든 공격 단계 포함)
- **조건**: `status = 'N'`인 데이터만 (중복 방지)

#### 수집 완료 확인
- **엔드포인트**: `POST /api/export/cve/confirm`
- **기능**: 수집 완료된 링크의 `status`를 'Y'로 업데이트
- **목적**: 다음 수집 시 중복 방지

#### Python 수집 모듈
- **파일**: `cve_api_collector.py`
- **기능**:
  - 5분 간격 자동 수집
  - 메시지 포맷팅
  - 카카오톡 Intent 제어 지원
  - status 자동 업데이트
- **설정**:
  - API_BASE_URL: 서버 주소
  - API_TOKEN: 인증 토큰
  - PAGE_SIZE: 수집 개수
  - REPEAT_INTERVAL: 반복 간격

## 💻 기술 스택

### Backend (서버)
- **언어**: Node.js (v14 이상)
- **프레임워크**: Express.js
- **데이터베이스**: MySQL 8.0
- **인증**: JWT (JSON Web Token)
- **실시간 통신**: Socket.IO
- **보안**: bcrypt (비밀번호), DOMPurify (XSS), Multer (파일 업로드)
- **이메일**: Nodemailer (SMTP)
- **프로토콜**: HTTP (32577), HTTPS (32578)

### Frontend (클라이언트)
- **언어**: JavaScript (React)
- **프레임워크**: React 18
- **UI 라이브러리**: Material-UI (MUI)
- **라우팅**: React Router DOM
- **HTTP 클라이언트**: Axios
- **실시간 통신**: Socket.IO Client
- **빌드 도구**: Vite

### Python 수집 시스템
- **언어**: Python 3.8 이상
- **데이터베이스**: MySQL Connector
- **HTTP 클라이언트**: requests
- **GitHub API**: PyGithub
- **번역**: deep-translator (Google Translate)
- **AI**: gemini-cli (Google Gemini Pro)
- **파일 처리**: zipfile, os, shutil

### 데이터베이스
- **DBMS**: MySQL 8.0
- **인코딩**: UTF-8
- **테이블**: 10개
  - Github_CVE_Info
  - CVE_Info
  - CVE_Packet_AI_Analysis
  - CVE_Integrated_Data
  - users
  - board_posts
  - chat_messages
  - api_tokens
  - 기타

### 외부 서비스
- **GitHub API**: CVE 저장소 검색
- **CVE API**: cve.circl.lu
- **Google Gemini Pro**: AI 분석
- **Google Translate**: 번역
- **Gmail SMTP**: 이메일 전송
