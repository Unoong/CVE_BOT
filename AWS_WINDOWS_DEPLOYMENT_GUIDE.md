**비용 비교:**
- gp3 SSD: $0.08/GB/월
- st1 HDD: $0.045/GB/월 (약 44% 저렴)
- sc1 HDD: $0.015/GB/월 (약 81% 저렴, 하지만 매우 느림)

**추천 구성:**
- 시스템 + DB: gp3 SSD (필수)
- 백업/로그: st1 HDD (비용 절감)
- 아카이브: sc1 HDD 또는 S3 Glacier (장기 보관)

### **보안 그룹 설정**
```
인바운드:
- HTTP (80): 0.0.0.0/0
- HTTPS (443): 0.0.0.0/0
- RDP (3389): 본인 IP만
- MySQL (3306): 인스턴스 내부만

아웃바운드:
- 모든 트래픽 허용
```

### **스토리지 구성 (하이브리드 - 비용 절감)**
```
루트 볼륨: 50GB gp3 SSD (시스템) - 필수
데이터 볼륨: 50GB gp3 SSD (데이터베이스, 자주 접근하는 파일) - 필수
백업 볼륨: 100GB st1 HDD (백업, 로그, 오래된 CVE 데이터) - 선택
자동 스냅샷: 일 1회, 7일 보관
```

**스토리지 타입 선택 가이드:**
- **gp3 SSD (필수)**: 시스템, MySQL DB, 활성 사용자 파일
- **st1 HDD (권장)**: 백업 파일, 로그 파일, 오래된 CVE 아카이브
- **sc1 HDD (선택)**: 1년 이상 접근 안 한 데이터 (매우 저렴하지만 느림)
