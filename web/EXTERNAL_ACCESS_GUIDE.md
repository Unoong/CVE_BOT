# 외부 접속 설정 가이드

## 📋 개요
CVE Bot 웹 애플리케이션을 localhost가 아닌 외부 네트워크에서 접속할 수 있도록 설정하는 방법입니다.

---

## ✅ 완료된 설정

### 1. 백엔드 서버 (Node.js)
- `0.0.0.0`으로 바인딩하여 모든 네트워크 인터페이스에서 접근 가능
- 서버 시작 시 자동으로 로컬 및 외부 접속 주소 표시

### 2. 프론트엔드 (React + Vite)
- Vite 개발 서버를 `0.0.0.0`으로 설정
- API URL을 동적으로 설정하는 config 시스템 구현

## 🌐 외부에서 접속하기

### 같은 네트워크(LAN) 내에서 접속

1. **스마트폰이나 다른 PC에서:**
   - 브라우저 열기
   - 주소창에 입력: `http://192.168.0.100:3000` (서버 시작 시 표시된 Network 주소)
   - 정상적으로 웹사이트가 로드됩니다!

2. **다른 사람과 공유:**
   - 같은 Wi-Fi나 네트워크에 연결된 사람에게 주소 공유
   - 예: `http://192.168.0.100:3000`

### 인터넷(외부 네트워크)에서 접속하려면

⚠️ **추가 설정 필요:**

1. **공유기 포트 포워딩 설정:**
   - 공유기 관리 페이지 접속 (보통 192.168.0.1 또는 192.168.1.1)
   - 포트 포워딩/가상 서버 설정
   - 외부 포트: 3000 → 내부 IP: 192.168.0.100, 내부 포트: 3000
   - 외부 포트: 32577 → 내부 IP: 192.168.0.100, 내부 포트: 32577

2. **공인 IP 확인:**
   - https://www.myip.com 접속
   - 표시된 IP 주소 메모 (예: 203.0.113.45)

3. **외부에서 접속:**
   - `http://203.0.113.45:3000`

4. **보안 주의사항:**
   - 인터넷에 공개할 경우 HTTPS 사용 권장
   - 방화벽 설정 확인
   - 강력한 비밀번호 사용

## 🔧 설정 파일 위치

### 백엔드 API URL 자동 설정
```
web/client/src/config.js
```

이 파일이 자동으로 API URL을 설정합니다:
- 개발 모드: `http://localhost:32577/api`
- 프로덕션 모드: 현재 호스트 기반 자동 설정

### 수동으로 API URL 변경하려면

`web/client/src/config.js` 파일 수정:

```javascript
// 현재 (자동 설정)
export const API_URL = getApiUrl();

// 수동 설정으로 변경하려면 아래 줄의 주석 해제
export const API_URL = 'http://192.168.0.100:32577/api';
```

## 🔍 문제 해결

### "ERR_CONNECTION_REFUSED" 에러가 나는 경우

1. **서버가 실행 중인지 확인**
   ```bash
   # 백엔드
   node server.js
   
   # 프론트엔드
   npm run dev
   ```

2. **방화벽 확인**
   - Windows 방화벽에서 포트 3000, 32577 허용
   - 설정 → 방화벽 → 고급 설정 → 인바운드 규칙

3. **IP 주소 확인**
   ```bash
   # Windows
   ipconfig
   
   # 이더넷/WiFi 어댑터의 IPv4 주소 확인
   ```

### 외부에서 접속이 안 될 때

1. **같은 네트워크인지 확인**
   - 같은 Wi-Fi에 연결되어 있는지 확인

2. **공유기 설정 확인**
   - 공유기가 AP 격리(AP Isolation) 기능을 사용하는지 확인
   - 게스트 네트워크가 아닌 메인 네트워크 사용

3. **Windows 네트워크 프로필 확인**
   - 설정 → 네트워크 및 인터넷 → Wi-Fi → 네트워크 프로필
   - "개인" 또는 "사설"로 설정 (공용 아님)

## 📱 모바일에서 접속하기

### 1단계: PC와 같은 Wi-Fi 연결
- 스마트폰을 PC와 동일한 Wi-Fi에 연결

### 2단계: IP 주소 확인
- PC에서 서버 시작 시 표시된 Network 주소 메모
- 예: `http://192.168.0.100:3000`

### 3단계: 모바일 브라우저에서 접속
- Safari, Chrome 등 브라우저 열기
- 주소창에 입력: `192.168.0.100:3000`
- 엔터/이동

### 홈 화면에 추가하기 (선택)
- **iPhone (Safari):**
  1. 공유 버튼 탭
  2. "홈 화면에 추가" 선택
  
- **Android (Chrome):**
  1. 메뉴 (⋮) 탭
  2. "홈 화면에 추가" 선택

## 💡 팁

### 고정 IP 설정 (권장)
서버 PC의 IP가 자주 바뀌는 경우:

1. **공유기에서 DHCP 예약 설정**
   - 공유기 관리 페이지 접속
   - DHCP 설정 → IP 주소 예약
   - PC의 MAC 주소에 고정 IP 할당 (예: 192.168.0.100)

2. **PC에서 고정 IP 설정**
   - 제어판 → 네트워크 및 공유 센터
   - 어댑터 설정 변경
   - 네트워크 어댑터 우클릭 → 속성
   - IPv4 → 속성 → 수동 IP 설정

### QR 코드로 쉽게 공유
1. https://www.qr-code-generator.com 접속
2. URL 입력: `http://192.168.0.100:3000`
3. QR 코드 생성
4. 모바일에서 QR 코드 스캔하여 바로 접속
