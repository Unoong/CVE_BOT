# 🎨 로고 설정 가이드

## 로고 이미지 저장 방법

제공된 파란색 S 로고 이미지를 이 폴더에 `logo.png`라는 이름으로 저장하세요.

### 📍 저장 경로
```
E:\LLama\pythonProject\CVE_BOT\web\client\public\logo.png
```

### 📝 단계별 가이드

1. **이미지 다운로드**
   - 채팅에서 제공된 로고 이미지를 다운로드하세요

2. **파일 저장**
   - 다운로드한 이미지를 `E:\LLama\pythonProject\CVE_BOT\web\client\public\` 폴더에 저장
   - 파일명을 **정확히** `logo.png`로 설정

3. **확인**
   - 파일 경로: `E:\LLama\pythonProject\CVE_BOT\web\client\public\logo.png`
   - 브라우저에서 `http://localhost:3000/logo.png`로 접속하여 이미지가 보이는지 확인

4. **새로고침**
   - 웹사이트를 새로고침하면 로고가 표시됩니다
   - Ctrl + F5 (강제 새로고침) 권장

### 🎯 로고가 표시되는 위치

- **로그인 페이지**: 상단 중앙
- **회원가입 페이지**: 상단 중앙
- **메인 헤더**: 좌측 상단
- **사이드바**: 최상단
- **파비콘**: 브라우저 탭

### 📏 권장 이미지 사양

- **형식**: PNG (투명 배경 권장)
- **크기**: 512x512 픽셀 이상
- **파일 크기**: 500KB 이하
- **배경**: 투명 또는 흰색

### 🔧 문제 해결

#### 로고가 안 보일 때
1. 파일명이 정확히 `logo.png`인지 확인
2. 파일이 `public` 폴더에 있는지 확인
3. 브라우저 캐시 삭제 (Ctrl + Shift + Delete)
4. 개발 서버 재시작

#### 로고가 깨져 보일 때
- 이미지 파일이 손상되지 않았는지 확인
- PNG 형식인지 확인
- 이미지 뷰어로 정상 표시되는지 확인

### ✅ 저장 완료 후

저장 완료 후 아래 명령으로 개발 서버를 재시작하세요:

```bash
cd E:\LLama\pythonProject\CVE_BOT\web\client
npm run dev
```

로고가 정상적으로 표시됩니다! 🎉
