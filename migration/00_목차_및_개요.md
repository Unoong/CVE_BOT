## 시스템 한줄 요약

- **Python**: GitHub/CIRCL/CVE API 기반 CVE·POC 수집, Gemini AI 분석, DB 적재, 통합 데이터 생성
- **Node.js 웹**: 대시보드, CVE/POC 조회, 게시판, 채팅, 관리자, API 토큰, Export API
- **DB**: MySQL (TOTORO). Github_CVE_Info, CVE_Info, CVE_Packet_AI_Analysis, CVE_Integrated_Data, users, board_posts, chat_messages, api_tokens 등
