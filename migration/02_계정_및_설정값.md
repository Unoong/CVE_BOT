## 5. Python 설정 파일 (config.json / config_loader)

- **위치**: 프로젝트 루트 `config.json` (기본). 환경변수 `CVE_BOT_ENV`로 `config.test.json`, `config.prod.json` 선택 가능.
- **주요 키**: `database` (host, port, user, password, database), `collection` (수집 옵션, `max_cve_per_item`, `cve_specific_limits` 등).
- **config_loader.py**: `ConfigLoader.get_current_env()`, `load_config()`, `get_config_file_path(env)`.

## 8. API 토큰 (Export API)

- DB 테이블 `api_tokens`: name, token(64자), permissions(JSON), expires_at, is_active, created_by 등.
- 헤더: `X-API-Token: <token>` 으로 `/api/export/*` 호출.
- 관리: 웹 관리자 → API 토큰 메뉴에서 생성/비활성화/삭제.

---

## 9. 사용자 역할 (users.role)

| 역할 | 설명 |
|------|------|
| `user` | 일반 조회, 게시판, 채팅 |
| `analyst` | user + DB 쿼리 실행 |
| `admin` | analyst + 사용자/역할 관리, 로그, API 토큰, 시스템 설정, Gemini 할당량 조회 |

- 기본 가입 역할: 시스템 설정에서 지정 가능(예: 기본 role `user`).
