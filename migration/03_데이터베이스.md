## 연결 정보 (server.js / Python config)

| 항목 | 값 |
|------|-----|
| 호스트 | `localhost` |
| 포트 | `7002` |
| 사용자 | `root` |
| 비밀번호 | `!db8354` |
| 데이터베이스명 | `TOTORO` |
| 문자셋 | utf8mb4 권장 |

---

## 테이블 목록 및 용도

| 테이블명 | 용도 |
|----------|------|
| **Github_CVE_Info** | GitHub에서 수집한 CVE POC 메타(링크, 제목, 작성자, CVE, readme, download_path, status, AI_chk, cve_info_status 등) |
| **CVE_Info** | CVE 상세(공식/API 기반): CVE_Code, state, dates, product, descriptions, CVSS, solutions, Response_data 등 |
| **CVE_Packet_AI_Analysis** | AI(Gemini) 분석 결과: link, step, packet_text, vuln_stage, expected_response, mitre_tactic/technique, snort_rule, remediation 등 |
| **CVE_Integrated_Data** | 위 3개 소스 통합 뷰( hash_key, github_*, cve_*, ai_* 컬럼) |
| **users** | 웹 사용자: username, password(bcrypt), name, phone, email, nickname, role, created_at |
| **board_posts** | 자유게시판: user_id, title, content, file_path, views, created_at, updated_at |
| **chat_messages** | 채팅 메시지: user_id, username, nickname, message, created_at |
| **api_tokens** | API 토큰: name, token, permissions(JSON), expires_at, is_active, created_by, last_used_at |
| **email_verifications** | 이메일 인증 코드(회원가입/비밀번호 재설정) |
| **gemini_quota_events** 등 | Gemini 할당량/이벤트 기록(웹에서 조회) |

---

## 주요 인덱스 (참고)

- **Github_CVE_Info**: idx_cve, idx_link, idx_AI_chk, idx_status, idx_cve_info_status
- **CVE_Info**: idx_CVE_Code, idx_CVSS_Serverity
- **CVE_Packet_AI_Analysis**: idx_link, idx_step
- **CVE_Integrated_Data**: idx_hash_key, idx_github_cve, idx_cve_code
- **users**: idx_username, idx_email
- **api_tokens**: idx_token, idx_is_active

---

## Python DB 모듈 (db_manager.py)

- `get_db_connection(config)` : 연결 반환
- `create_table(conn)` : Github_CVE_Info 생성/확인
- `create_cve_info_table(conn)` : CVE_Info 생성/확인
- `create_ai_analysis_table(conn)` : CVE_Packet_AI_Analysis 생성/확인
- `create_integrated_table(conn)` : CVE_Integrated_Data 생성/확인
- `check_cve_info_exists(conn, cve_code)`, `insert_cve_info(conn, cve_info)`
- `insert_cve_data(conn, ...)`, `get_cve_count(conn, cve_code)`
- `get_cve_info_pending(conn)`, `update_cve_info_status(conn, link, status)`
- `insert_integrated_data(conn, ...)` 등

상세 스키마는 프로젝트 루트 `DATABASE_STRUCTURE.md` 참고.
