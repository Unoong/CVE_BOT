# 11. 웹 대시보드

## 역할

- CVE/수집/AI 분석 통계를 한 화면에 표시. 차트, 요약 숫자, 필터.

## 프론트

- **페이지**: `client/src/pages/Dashboard.jsx`
- **라우트**: `/` 또는 `/dashboard` (프로젝트 라우팅에 따름)

## API

- **GET /api/dashboard/stats**: 전체 통계(총 CVE 수, POC 수, AI 분석 수, 위험도별 수, 최근 수집 트렌드 등).
- **GET /api/dashboard/collection-trends**: 수집 트렌드(일별/기간별).
- **GET /api/dashboard/detailed-stats**: 상세 통계(필요 시).
- **POST /api/dashboard/filter-cves**: 조건에 맞는 CVE 필터링(날짜, 위험도 등).

## 표시 항목 예

- 총 CVE 개수, POC 개수, AI 분석 완료 개수.
- 위험도별(Critical/High/Medium/Low) 건수.
- 수집 트렌드 차트(Recharts 등).
- 통계 집계 시점(마지막 갱신 시간).
- 필터: 기간, CVSS 구간, CWE 등(구현에 따라 다름).

## 통계 갱신

- **init_dashboard_stats.js**: 대시보드용 집계 테이블/뷰 갱신.
- **update_dashboard_stats.bat**: 위 스크립트 실행.
- **스케줄**: Windows 작업 스케줄러로 매일 오전 3시 등에 `register_task_scheduler.ps1`로 등록된 작업 실행.
