# 15. 웹 인증·프로필

## 회원가입

- **페이지**: Register.jsx
- **API**:
  - **POST /api/auth/send-verification**: 이메일로 인증 코드 전송.
  - **POST /api/auth/verify-code**: 인증 코드 확인.
  - **POST /api/auth/register**: username, password, name, phone, email, nickname, (기본 role은 시스템 설정에서 지정).

## 로그인

- **페이지**: Login.jsx
- **API**: **POST /api/auth/login** — username, password. 성공 시 JWT 반환.
- **GET /api/auth/me**: 토큰으로 현재 사용자 정보 조회.

## 비밀번호 찾기

- **페이지**: FindAccount.jsx 등
- **API**:
  - **POST /api/auth/find-id**: 이메일로 아이디 찾기.
  - **POST /api/auth/reset-password-send-code**: 재설정용 인증 코드 전송.
  - **POST /api/auth/reset-password-verify-code**: 코드 확인.
  - **POST /api/auth/reset-password**: 새 비밀번호로 변경.

## 이메일 발송

- **nodemailer**: Gmail (server.js 내 user/pass 또는 env). 회원가입 인증, 비밀번호 재설정 인증 코드 발송.
- 인증 코드는 메모리(Map/객체) 저장. 만료 시간 적용 권장.

## 프로필

- **페이지**: Profile.jsx
- **API**:
  - **GET /api/users/profile**: 본인 정보(이름, 이메일, 닉네임, 역할 등).
  - **PUT /api/users/profile/nickname**: 닉네임 변경.
  - **PUT /api/users/profile/password**: 비밀번호 변경(현재 비밀번호 확인 후).

## 관리자 비밀번호 초기화

- **web/reset_admin_password.js**: admin 계정 비밀번호를 지정 값으로 bcrypt 해시 후 DB 업데이트. 스크립트 내 새 비밀번호 설정 후 실행.
