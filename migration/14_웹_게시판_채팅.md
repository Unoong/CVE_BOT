# 14. 웹 게시판·채팅

## 자유게시판

- **페이지**: Board.jsx, BoardDetail.jsx, BoardWrite.jsx, BoardEdit.jsx
- **API**:
  - **GET /api/board/posts**: 목록(페이지, 검색 등).
  - **GET /api/board/posts/:id**: 상세, 조회수 증가.
  - **POST /api/board/posts**: 글 작성(인증), multipart 파일 첨부 가능.
  - **PUT /api/board/posts/:id**: 수정(본인/관리자).
  - **DELETE /api/board/posts/:id**: 삭제(본인/관리자).

## 보안

- **XSS**: 본문/제목은 DOMPurify 등으로 정제 후 저장·표시.
- **파일 업로드**: 확장자·MIME 검증, 저장 경로 `uploads/`, 고유 파일명(타임스탬프+난수+안전한 이름).

## 실시간 채팅

- **컴포넌트**: `client/src/components/ChatWidget.jsx`
- **전송**: Socket.IO로 메시지 전송. **POST /api/chat/upload-image** 로 이미지 업로드 후 URL을 메시지에 포함 가능.
- **저장**: 서버에서 `chat_messages` 테이블에 user_id, username, nickname, message, created_at 저장.
- **정리**: 3일 이상 된 메시지 자동 삭제(서버 시작 시 또는 주기 작업).
