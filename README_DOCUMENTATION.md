## 🔐 보안 주의사항

### 민감 정보 관리

이 문서들은 **비밀번호 등 민감한 정보를 마스킹**하여 작성되었습니다.

**마스킹된 정보**:
- ✅ MySQL 비밀번호: `****`
- ✅ Gmail 앱 비밀번호: `****`
- ✅ API 토큰 예시: `b36e...fcdac`
- ✅ 개인 연락처: `010-XXXX-XXXX`

**실제 운영 시**:
- 각 설정 파일에서 실제 비밀번호 확인
- `config.json`: DB 비밀번호, GitHub 토큰
- `web/server.js`: DB 비밀번호, 이메일 비밀번호

**보안 권장 사항**:
1. 비밀번호는 주기적으로 변경 (3개월마다)
2. API 토큰은 안전한 곳에 보관
3. `wets.txt` 파일은 보안 유지 (로그인 비밀번호 기록)
4. 데이터베이스는 외부 접근 차단
5. HTTPS 인증서는 Let's Encrypt로 교체 권장
